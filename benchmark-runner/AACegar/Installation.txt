1) Extract the source files into the working folder
2) Install mpfr and gmp
     apt-get install libgmp3-dev libmpfr-dev libmpfr-doc libmpfr4 libmpfr4-dbg 
3) Install Eigen 3.2.8 or later.
   Download the latest release of eigen from http://eigen.tuxfamily.org (the original was built with eigen 3.2.8) and unpack into ./Eigen
4) Install boost 1.59 or later: download boost. Unpack the library into ./boost
     wget -O boost_1_59_0.tar.gz http://sourceforge.net/projects/boost/files/boost/1.59.0/boost_1_59_0.tar.gz/download
     tar xzvf boost_1_59_0.tar.gz
     mv boost_1_59_0 boost
5) Make axelerator
     cd aa;make
   For help about usage and file format run axelerator and see the help provided.
